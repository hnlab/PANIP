# Benchmark Sets  
structures and energy of  
1. low-energy dimers
2. optimized low-energy dimers
3. Cambridge Structural Database (CSD) derived geometries
4. non-equilibrium conformations from random sampling  

can be download:  
```bash
wget https://zenodo.org/record/15514804/files/benchmark_sets.tar.gz
tar -xzvf benchmark_sets.tar.gz
```
